# Workspace Numerical Packages

The workspace virtual environment currently provides:

| Import | Use |
|---|---|
| `numpy` | scientific computing |
| `scipy` | algorithms for optimization, integration, interpolation, eigenvalue problems, algebraic equations, differential equations, statistics and many other classes of problems. |
| `sympy` | symbolic mathematics |
| `numba` | JIT compiler that translates a subset of Python and NumPy code into fast machine code |
| `torch` | tensor computation and autograd system |
| `mpmath` | real and complex floating-point arithmetic with arbitrary precision |
| `pandas` | data analysis and manipulation tool |
| `matplotlib` | library for creating static, animated, and interactive visualizations |
| `networkx` | creation, manipulation, and study of the structure, dynamics, and functions of complex networks |
| `pulp` | model and solve Linear Programming (LP) and Mixed-Integer Linear Programming (MILP) problems |
| `ortools` | solve combinatorial optimization problems |
| `highspy` | high performance serial and parallel solver for large scale sparse linear optimization problems |
| `pysat` | Boolean satisfiability (SAT) solvers as well as to a variety of cardinality and pseudo-Boolean encodings |
| `cvxpy` | model and solve convex optimization problems |
| `osqp` | convex quadratic programs (QPs) solver using a specialized ADMM-based first-order method |
| `clarabel` | interior point numerical solver for convex optimization problems using a novel homogeneous embedding |
| `sklearn` | machine learning algorithms and data analysis |
| `faiss` | efficient similarity search and clustering of dense vectors |
